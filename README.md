# Proyecto de Procesamiento de Lenguaje Natural


## Resumen
Este proyecto se divide en dos partes, cada una en un notebook distinto. Además, hay un fichero `utils` para funciones y demás extra que se cree conveniente tener separado del los notebooks, por mayor legibilidad. 

En el primer notebook se explora diferentes métodos para clasificar textos, mientras que en el segundo se pone en práctica un sistema RAG y agente para tener un enfoque distinto para clásificar.
